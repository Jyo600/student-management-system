# Student Management System

## 📌 Description
A simple Java-based Student Management System to add, view, search, and delete student records using ArrayList and menu-driven program.

## 🚀 Features
- Add student
- View students
- Search student
- Delete student
- Show total number of students

## 🛠️ Technologies Used
- Java
- ArrayList
- Scanner
- Switch-case

## ▶️ How to Run
1. Compile the program:
   javac studentsystem.java

2. Run the program:
   java studentsystem
